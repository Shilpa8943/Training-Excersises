Introduction :

CSS Text properties

  • font-size: 18px; → Sets the text size.
  • color: #007BFF; → Changes text color to blue.
  • line-height: 1.6; → Increases spacing between lines.
  • font-family: Arial, sans-serif; → Uses Arial as the primary font.
  • font-weight: bold; → Makes the text bold.
  • text-align: justify; → Justifies text alignment.

Box 1 : Background

The CSS background properties are used to add background effects for elements

CSS background properties :

  • background-color
  • background-image
  • background-repeat
  • background-position
  • background-attachment

Box 2 : Border

The CSS border properties allow you to specify the style, width, and color of an element's border.

CSS border properties :

  • border-style(solid,dashed,double,dottted,etc.)
  • border-width
  • border-color
  • border-radius

Box 3 : Margin and Padding

  • Margins are used to create space around elements, outside of any defined borders.
  • Padding is used to create space around an element's content, inside of any defined borders.

CSS margin and padding properties :

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Pseudo-class Example:

1) Hover : When the user hovers over an element with the mouse.

2) Active : When an element is clicked on.


Pseudo-Elements Example:

1)::before : The ::before pseudo-element can be used to insert some content before the content of an element.

2)::after : The ::after pseudo-element can be used to insert some content after the content of an element.
